# ScholarAI Research Assistant

AI-powered research synthesis tool that searches the web, analyzes sources, and generates comprehensive reports.

## Features
- Web search integration (Tavily/SerpAPI)
- AI-powered synthesis with OpenAI
- Multiple export formats (Markdown, JSON)
- Customizable writing styles and tones

## Usage
1. Enter your research topic
2. Configure number of sources and style preferences
3. Click "Start Research"
4. Review and download your synthesized report
